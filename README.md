# BudgetBuddy
A friendly AI bot that helps you with prices


Team Info: Ali Khan, Abdul-Wahab & Mohammed Tabraiz Siraj
Project: BudgetBuddy - Your friend for staying on a budget

What It Does: 
We have 7 main Features:

1) BuddyBudget ChatBot: An AI-powered shopping assistant powered by Ollama 3.0. It can track prices across multiple stores, help you find the cheapest items, and compare products from Costco, Walmart, and Loblaws. If you go off-topic, it gently steers you back to grocery shopping and can assist with any shopping-related queries.
2) Account Authentication Login: It allows you to secure login and signup system using via Firebase. Firebase will store your email ensuring a smooth and secure account management process.
3) Product Search DashBoard: A central dashboard to search for a wide variety of products, track their current prices, and explore their historical price trends.
4) Store Price Comparison: Compares prices of multiple items across Walmart, Loblaws, and Costco, helping users identify the best deals and save money. it will be green for the cheapest item. It is done via a bar graph.
5) Budget Insights: 
Provides detailed information on price change over time, highlighting if prices have increased since last change in Walmart, Loblaws, and Costco.
6) Price Trend Tab: Displays average price history for products in a line graph. There is also a linear graph that shows the trend showing the lines gradient to show how inflation is effecting the trend of the market. It helps visualise changes in inflation.
7) Category Trends: Shows price trends for major categories including Electronics, Groceries, Household items, and Personal-Care products, enabling smarter shopping decisions.

How to Run It:
1) We clone the repository
2) install python extension and application e.g. live server, flask
3) set up ollama
4) prepare dataset
5) Then we run the application via login.html (using live server)


Apps you need to install:
1) Python 3.7 or above
2) Flask
3) Ollama with Llama 3.0 model
4) Firebase account for authentication, Live Server.

Impact: 
1) Cost Savings: identifies best deals across major retailers (Costco, Walmart, etc.)
2) Helps in decision making via cost-time graphs
3) Accessibility: User-friendly interface, accessible to shoppers of all technical levels
4) Time Efficient


Assumptions: 
1) Price data is updated regularly and accurately reflects current market prices
2) The dataset covers a representative sample of commonly purchased items
3) Three major retailers (Walmart, Costco, Loblaws) represent the primary shopping options for most users
